## Задание 1 - Оценка ПК для пула облачного гейминга

Условие:

Необходимо описать подход к оценке ПК, которые потенциально могут быть включены в пул облачного гейминга, и определить, подходит ли такой ПК для современных игр и комфортного игрового опыта.

В ответе требуется:

1. Указать, на какие параметры проводится оценка (и почему они важны)

2. Описать способы и инструменты проверки

3. Указать источники информации (требования игр, драйверы, обновления, совместимость и т.д.)

4. Оформить пример проверки двух ПК в виде таблицы:

- один ПК проходит проверку

- один ПК не проходит (с указанием причины)

Реальные тесты проводить не требуется — важны логика и подход к оценке.

**Важно учесть:** ПК будут использовать игроки и поэтому важен комфортный игровой опыт

Решение находится в папке [Task_1](./Task_1)

## Задание 2 - Python-скрипт мониторинга загрузок Steam

Условие:

Необходимо написать Python-скрипт, работающий в фоне и отслеживающий скорость загрузки игр в Steam независимо от места установки лаунчера.

Скрипт должен уметь:

1. Считывать скорость загрузки

2. Выводить скорость загрузки каждую минуту (в течение 5 минут)

3. Учитывать статус загрузки (например, пауза)

4. Дополнительно выводить, какая игра скачивается в данный момент

Подсказка: информацию о статусе загрузок можно получить из логов Steam, а путь установки — из реестра.

Решение находится в папке [Task_2](./Task_2)

## Задание 3 - C# консольное приложение (работа с IP)

Условие:

Необходимо написать консольное приложение на C#.

Требуется:
1. Загрузить все IP-адреса из файла в память (из файла [IPs.txt](./Task_3/Task_3_IPAnalytics/IPs.txt))

2. Создать функцию, выполняющую GET-запрос к https://ipinfo.io/{ip}/json и возвращающую объект IpData

3. Определить, IP каких стран представлены в файле, и их количество

4. Найти страну с максимальным числом IP и вывести города для этой страны

Ограничения:

- Разрешено использовать только Newtonsoft.Json

- Все необходимые значения должны вычисляться кодом

Решение находится в папке [Task_3](./Task_3)
